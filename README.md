## 📝 Description

**Ninja_Secure_Chat** is a sophisticated terminal-based IRC client designed for secure, decentralized, and anonymous communication. Built on the powerful Python-IRC engine, it allows users to connect to global servers (Libera Chat, Rizon, Undernet) without the complexity of traditional IRC commands. With a built-in auto-installer and a dual-pane UI, it's the perfect tool for private group discussions and secure team coordination.

## ✨ Key Features

* ⚙️ **Auto-Dependency Installer:** No more `ModuleNotFoundError`. The script automatically detects and installs missing libraries (irc).
* 🌐 **Multi-Server Support:** Pre-configured with the world's most stable IRC networks: Libera Chat, Freenode, Undernet, DALnet, and Rizon.
* 🖥️ **Dual-Pane Terminal UI:** Interactive split-screen display showing chat history on the left and a command guide on the right.
* 🔐 **Private Channels:** Full support for password-protected channels (Keys) for end-to-end authorized access.
* 🔔 **Real-Time Presence:** Instant terminal notifications when users join, quit, or change their nicknames.
* ⚡ **WAN Ready:** Works globally over the internet without the need for port forwarding or complex VPN setups.

* **Active Internet Connection** (WAN)

## 🚀 Installation & Usage

To deploy **Ninja_Secure_Chat**, follow these steps in your terminal:

```bash
# 1. Clone this repository
git clone https://github.com/CyberNinja-Tz/Ninja_Secure_Chat

# 2. Navigate to the directory
cd Ninja_Secure_Chat

# 3. Launch the tool like a Ninja
python3 Ninja_Secure_Chat.py

```
### 📂 Architecture

All communication is transient. Ninja_Secure_Chat does not store logs on a central server, ensuring your conversations stay between you and your peers.
